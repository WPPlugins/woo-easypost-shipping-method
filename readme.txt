Easypost Shipping Method Plugin for WooCommerce. Displays Live Shipping Rates from USPS, UPS and FedEx based on the Shipping Address and Cart Content.



== Description ==



= Introduction =



EasyPost WooCommerce Shipping extension with USPS, UPS and FedEx Postage helps WooCommerce shops to streamline EasyPost shipping integration by providing real-time shipping rates. Based on the postal codes and other parameters, all available shipping services along with the rates are listed for the customers to choose from.



= Integrates WooCommerce to EasyPost =



Once this plugin is installed and configured with necessary information (please visit installation section for more info), your WordPress/WooCommerce Shop will be ready to ship using EasyPost. This plugin will add EasyPost shipping method as one of the shipping methods in WooCommerce.



= Calculate shipping rates dynamically =



While checking out, a customer is presented with the available shipping services and the rates based on the shipping address and cart content. Customer can choose the best method that matches his/her requirements and proceed to payment.

== Installation ==



1. Upload the plugin folder to the ‘/wp-content/plugins/’ directory.

2. Activate the plugin through the ‘Plugins’ menu in WordPress.

3. Thats it – you can now configure the plugin.



== Frequently Asked Questions ==



= After the setup, my site is still saying “There doesn‘t seem to be any available shipping methods”. =



You can enable debug in settings and debug messages will start showing up in cart/checkout page, from which you will get a clue what is the issue about. Hope EasyPost shipping method is enabled. Product dimensions and weight is configured correctly.
